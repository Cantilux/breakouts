Breakouts Framework Demo

.container

A centered content wrapper with a fixed max-width and responsive side padding.

I'm inside a .container.

.full-bleed

Expands the element to the full width of the viewport, ignoring any container boundaries.

This is a .full-bleed section.

.breakout

Breaks the element out of the container's padding but still keeps margins on both sides.

This is a .breakout section.

.grid

A CSS grid with three columns. Content is centered in the middle column.

This section is in the .grid's main area.

.grid.grid--full

When used with .grid--full, children will expand to fill the entire grid width.

This section spans the full width using .grid--full.

.breakouts-grid

A grid layout using named lines like content, popout, feature, full.

.content — centered content
.popout — slightly wider
.feature — visual emphasis
.full — full-width
