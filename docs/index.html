Breakouts Framework Demo

.container

A centered content wrapper with a fixed max-width and responsive side padding.

I'm inside a .container.

.full-bleed

Expands the element to the full width of the viewport, ignoring any container boundaries. Useful for hero sections or background elements.

This is a .full-bleed section.

.breakout

Breaks the element out of the container's padding but still keeps margins on both sides. Great for wide callouts or media elements.

This is a .breakout section.

.grid

A CSS grid with named lines: main and full. Default children will stay within the main column.

This section is in the .grid's main area.

.grid.grid--full

When used with .grid--full, children will expand to fill the entire grid width. Useful for background or full-width rows inside a structured layout.

This section spans the full width of the grid using .grid--full.

.breakouts-grid

A full grid layout using named lines such as content, popout, feature, and full. Useful for building precise and responsive layouts.

.content — Centered main content area
.popout — Slightly outside the content width
.feature — Wider area, great for visual emphasis
.full — Full width across the viewport
